## 简介

本仓库用于同步 Kimi Work 的 Skills 文件，方便团队成员共享和备份技能资产。

## 仓库结构

```
sparkcon-skills/
├── README.md              # 本文件
├── skills/                # Managed Skills（本地管理）
│   ├── sparkcon/          # ⭐ Spark 黑客松专用 Skill
│   ├── swarm-coding/      # 蜂群编码协作
│   ├── obsidian-sync/     # Obsidian 同步
│   └── ...
└── plugin-skills/         # Plugin 提供的 Skills
    ├── audio_generation/
    ├── scholar/
    └── ...
```

## 快速使用

### 克隆仓库

```bash
git clone https://github.com/Jup33Q/sparkcon-skills.git
```

### 同步最新 Skills

```bash
cd sparkcon-skills
# 拉取最新变更
git pull origin main
```
